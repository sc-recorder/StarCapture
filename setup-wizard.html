Setup Wizard

Welcome to StarCapture!

Let's set up your recording environment for Star Citizen. This wizard will help you:

  • Install or verify OBS Studio (our recording backend)
  • Detect your hardware encoders for optimal performance
  • Configure your audio devices and voice applications
  • Set up your recording resolution
  • Generate optimized recording profiles

This process will only take a few minutes and ensures the best recording experience.

Star Citizen Installation

Select your Star Citizen installation folder to enable event detection from game logs.

Select the folder containing LIVE and/or PTU folders (e.g., C:\Program Files\Roberts Space Industries\StarCitizen)

ℹ️ This allows the recorder to detect in-game events like kills, deaths, and quantum jumps.
You can switch between LIVE and PTU builds from the main interface.
